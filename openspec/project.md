# Project Context

## Purpose
- tiles是一个基于golang开发的工具包集合，主要用来协助构建微服务项目。
- tiles内封装了日志（log），上下文(context),配置中心（config）,网关（gateway）,微服务注册中心（Service Registry）等等常用工具，并且每一种工具可以单独引用，不必导入整个tiles项目。
- tiles的愿景正如他的名字一样，希望能够为你搭建你的应用贡献一点微薄的力量
